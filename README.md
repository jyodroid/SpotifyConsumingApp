# SpotifyConsumingApp
This project consumes Spotify appy using kaaes (https://github.com/kaaes/spotify-web-api-android). 
Aditionally the Libraries used:
  +ButterKnife
  +Picasso

# Intalation Guide
  +enable USB debugging on phone(Settings->application-> Development -> USB Debugging), 
  +install the Android USB device driver 
  +plug the phone into your computer.
  +use adb utility to install the apk (adb install <path-to-your-APK>)
  for adb utility download look at http://developer.android.com/tools/help/adb.html

# User Guide
  +Use the text field to write the name of an artist
  +Scroll down to explore albums from this artist
  +select one album and touch over
  +touch the button to go to spotify album page
